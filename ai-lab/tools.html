AI Tools

Tools I use (or have used) daily for local AI experimentation and development

☁️ Cloud Models

✨ Gemini

Purpose: General assistant, multimodal tasks

Google's AI model with strong multimodal capabilities. Good for general tasks, image understanding, and long context windows.

Why I use it: Free tier available, good for quick questions and image analysis.

💬 ChatGPT

Purpose: General assistant, code generation

OpenAI's conversational AI. Widely used for general tasks, coding, writing, and brainstorming.

Why I use it: Good baseline for comparison, strong coding capabilities.

Visit ChatGPT →

🧠 Claude

Purpose: Long context, nuanced reasoning

Anthropic's AI model known for helpful, harmless, and honest responses. Excellent at long-form content and nuanced analysis.

Why I use it: Strong reasoning, good at following complex instructions.

Visit Claude →

🔍 DeepSeek

Purpose: Code generation, reasoning

Chinese AI lab with strong open-source models. Known for efficient training and good coding capabilities.

Why I use it: Competitive performance, good for code tasks.

🌙 Kimi

Purpose: Long context, document analysis

Moonshot AI's model with very long context windows. Good for analyzing large documents and multi-turn conversations.

Why I use it: Handles long contexts well, good for document analysis.

Visit Kimi →

🔎 Perplexity

Purpose: Search-augmented answers

AI-powered search engine that provides cited answers. Good for research and finding current information.

🤖 Agent Workspaces

🦙 Llama.cpp

Purpose: Local LLM inference engine

C/C++ based LLM inference engine. The foundational runtime that powers many local AI tools.

Why I use it: Exposes all llama.cpp features as soon as they release vs awaiting often later release for a wrapper in tools such as Ollama and LM Studio.

Visit Llama.cpp →

🔄 Llama-swap

Purpose: Model swapping and management

Tool for quickly swapping between different LLM models without reloading from scratch.

Why I use it: Lets me switch between models seamlessly during testing and comparison workflows.

Visit Llama-swap →

🤖 MiMoCode

Purpose: AI-assisted development

Interactive CLI tool for software engineering tasks. Helps with code editing, debugging, and project navigation.

Why I use it: Fast, understands context well, works with local models.

Visit MiMoCode →

🔧 Nanobot

Purpose: Lightweight AI agent framework

Minimal agent framework for running AI-powered tasks locally.

Why I use it: Simple setup for quick agent experiments.

Visit Nanobot →

⚙️ Picoclaw

Purpose: Micro agent tooling

Small-scale agent utilities for local AI workflows.

Why I use it: Useful for lightweight automation tasks.

🔩 Moltis

Purpose: AI workflow orchestration

Tool for orchestrating multi-step AI workflows locally.

Why I use it: Helps chain together model calls for complex tasks.
